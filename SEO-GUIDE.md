## 📝 内容优化清单

### 每篇文章检查项

1. **标题优化**
   - [ ] 包含目标关键词
   - [ ] 长度 15-30 字
   - [ ] 有吸引力（数字、疑问、方法）

2. **正文优化**
   - [ ] 首段包含关键词
   - [ ] H2/H3 标题包含关键词
   - [ ] 关键词密度 1-3%
   - [ ] 添加 2-3 个内部链接

3. **结构优化**
   - [ ] 有目录 (TOC)
   - [ ] 段落不超过 5 行
   - [ ] 使用列表和表格
   - [ ] 代码块有语法高亮
